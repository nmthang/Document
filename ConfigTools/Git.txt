*Push
	git push -u origin master
* Pull
	git pull origin master
*List the remote connections you have to other repositories.
	git remote -v
*Add: git remote add <name> <url>
*Remove: git remote rm <name>
*Rename: git remote rename <old-name> <new-name>
*Fetch: git fetch <remote>
		git fetch <remote> <branch>

		
*Branch:
	+ List all branch: git branch
	+ Create new branch: git branch <new_name_branch>
	+ Create & checkout: git checkout -b <new-branch>
	+ Delete: git branch -d <name>
